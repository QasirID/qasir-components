# Badge Variations

Small notification like UI Element, used to catch user attention to a certain elements.

```javascript
import { Badge } from 'qasirc';
```

<br />

```html
<Badge showWhenZero={true} />
```
<br />
```html
<Badge showWhenZero={false} />
```
